android string.xml知识点

为什么使用string.xml 存储文本

  • 国际化
  • 格式化
  • 表达复数
  • 单词高亮
  • 减小体积，降低冗余

1、国际化

android应用的国际化，是在src目录下创建不同语言的values目录。并在目录中创建string.xml。不同目录的string.xml中实现国际化。

2、格式化

使用 Resources#getString(int id, Object…formatArgs) 来格式化字符。尽量避免使用 + 来连接字符串，因为在不同的语言里对于词汇的划分是不一致的。
