Win10 Git 配置SSH

当我们使用github或者码云等仓库时我们有可能需要ssh认证，所以需要生成他的ssh key。

1、安装GitBash

1、首先你要安装git工具
下载地址：https://git-scm.com/downloads

2、生成SSH keyPair

1、右键鼠标，选中 “Git Bash here”

2、输入指令，进入.ssh文件夹

如果提示 “ No such file or directory”，你可以手动的创建一个 .ssh文件夹即可

3、配置全局的name和email，这里是的你github或者bitbucket的name和email

查看设置

结果如图

使用如下命令可以取消全局设置：

3、Git生成单个SSH key及管理生成的多个SSH Key

连续按三次回车，这里设置的密码就为空了，并且创建了key。

生成两个文件：id_rsa和id_rsa.pub

3、 github、 gitLab 或者 码云 配置SSH

1、查看SSH公钥,并复制公钥内容


打开码云个人设置SSH,复制后保存即可愉快的使用git ssh 了
