Ubuntu16.4 安装PHP运行环境 LAMP

1、安装apache

测试： 浏览器访问http://Ubuntu的IP（即http://localhost），出现It Works!网页。
查看状态： service apache2 status/start/stop/restart
Web目录：/var/www
安装目录： /etc/apache2/
全局配置： /etc/apache2/apache2.conf
监听端口： /etc/apache2/ports.conf
虚拟主机： /etc/apache2/sites-enabled/000-default.conf

2、安装mysql

测试：mysql -u root -p
查看状态：service mysql status/start/stop/retart
查看监听端口的情况：netstat -tunpl或 netstat -tap

然后整合一下php和mysql

3、安装php7.0

apt-get安装

输入版本信息，检测安装是否成功
php -v
PHP 7.0.4-7ubuntu2.1 (cli) ( NTS )
Copyright (c) 1997-2016 The PHP Group
Zend Engine v3.0.0, Copyright (c) 1998-2016 Zend Technologies
with Zend OPcache v7.0.6-dev, Copyright (c) 1999-2016, by Zend Technologies

整合一下php和apache,安装apache2 php7 module

4、验证

重启MySQL和Apache

测试Apache能否解析PHP
vim /var/www/html/phpinfo.php
文件中写：

安装：选择apache2，点击确定。下一步选择是要配置数据库，并输入密码。
创建phpMyAdmin快捷方式：sudo ln -s /usr/share/phpmyadmin /var/www/html
启用Apache mod_rewrite模块：sudo a2enmod rewrite
重启服务：
service php7.0-fpm restart
service apache2 restart
测试：浏览器访问：http://ubuntu地址/phpmyadmin(即http://localhost/phpmyadmin)

6、配置Apache

添加：
AddType application/x-httpd-php .php .htm .html
AddDefaultCharset UTF-8
重启Apache服务

###6、解决常见问题

####1、ubuntu 16.04 可能乱码的问题
解决中文乱码：

修改为
LANG=”zh_CN.UTF-8”
LANGUAGE=”zh_CN:zh”
然后重启

最后 安装中文包

####2、 xml_parser_create() 不支持的问题
解决这个问题可以在这里
XML Parser
安装XML Parser

之后重启apache.
